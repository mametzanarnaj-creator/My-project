# Infrastructure

Бұл папкада жобаның инфрақұрылымын сипаттайтын файлдар орналасады.

## Қолданылатын технологиялар
- Docker — қосымшаны контейнерлеу
- Kubernetes — контейнерлерді басқару
- Terraform — инфрақұрылымды код арқылы басқару (IaC)

## Инфрақұрылым сипаттамасы
- Backend және Frontend Docker контейнерлерінде жұмыс істейді
- Қосымша Kubernetes кластерінде орналастырылады
- Деректер базасы бұлттық серверде орналасады

## CI/CD
- Код репозиторийге жіберілген кезде автоматты түрде тексеріледі
- Docker бейнесі жасалып, Registry-ге жүктеледі
- Қосымша Staging ортасына орналастырылады
